# Thunder Connector

A node module to connect to the USB [Thunder Missile Launcher](http://www.dreamcheeky.com/thunder-missile-launcher). 


## Dependencies

### libusb

At first you have to install [libusb](http://www.libusb.org/) in order to use this module.
